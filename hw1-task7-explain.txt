Для создания alias для команды ls -la была выполнена команда alias ll='ls -a'
Для того, чтобы сделать alias постоянным, как и в случае с переменной PATH из задачи 2,
следует добавить команду в файл .bashrc и сохранить изменения, перезапустив терминал или через source.
Команды для сохранения изменений в .bashrc:
echo "alias ll='ls -la'" >> ~/.bashrc
source ~/.bashrc
