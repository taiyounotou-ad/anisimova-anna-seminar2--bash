Для создания alias для команды ls -la была выполнена команда alias ll='ls -a'
Для того, чтобы сделать alias постоянным, как и в случае с переменной PATH из задачи 2,
следует добавить команду в файл .bashrc и сохранить изменения, перезапустив терминал или через source.
Команды для сохранения изменений в .bashrc:
echo "alias ll='ls -la'" >> ~/.bashrc
source ~/.bashrc
После перезапуска терминала сохраненный alias успешно отобразился
Чтобы использовать автодополнение, после ввода команды cd были введены первые буквы имени файлов 'hw' и нажата кнопка Tab. Автоматически добавилась общая часть имен файлов 'hw1-task', а при двойном нажатии Tab был выведен список всех файлов, в имени которых присутсвует 'hw1-task'.
